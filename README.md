# UPI Reliability Intelligence Dashboard

## Overview

This project analyzes **250,000 UPI transaction records** using Microsoft Excel to monitor transaction activity across payment apps, banks, networks, and time periods.

The objective is to transform raw transaction data into an interactive dashboard that enables users to explore transaction trends, monitor key performance indicators (KPIs), and gain insights into digital payment activity.

---

## Business Problem

Millions of UPI transactions occur every day, making it difficult to manually monitor transaction activity and identify usage patterns.

This dashboard provides a centralized view of transaction metrics, helping stakeholders understand transaction distribution, peak activity periods, and transaction outcomes.

---

## Project Objectives

* Analyze 250,000 UPI transaction records
* Monitor transaction activity across payment apps and banks
* Track transaction success and failure distribution
* Identify peak transaction hours
* Build an interactive Excel dashboard using Pivot Tables and Slicers

---

## Dashboard Features

* KPI Cards

  * Total Transactions
  * Successful Transactions
  * Failed Transactions
  * Success Rate
  * Failure Rate

* Interactive Charts

  * Transaction Activity by Hour
  * Transaction Activity by Payment App
  * Transaction Activity by Bank
  * Transaction Activity by Network
  * Success vs Failure Distribution

* Interactive Slicers

  * Payment App
  * Bank
  * Network Type
  * Amount Category

---

## Key Insights

* Analyzed 250,000 UPI transactions.
* Overall transaction success rate is approximately 95%.
* Transaction activity peaks during evening hours.
* PhonePe and Google Pay process the highest transaction volumes.
* SBI handles the largest transaction volume among banks.
* 4G users account for the highest transaction activity.
